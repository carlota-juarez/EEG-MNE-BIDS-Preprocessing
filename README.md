# MNE-BIDS-Pipeline [1]: MEEG Preprocessing

This Brainlife App imports and preprocesses M/EEG data (EDF for EEG and FIF for MEG) using MNE-BIDS-Pipeline. The App converts the input data into Brain Imaging Data Structure (BIDS) format and runs the init and preprocessing steps of MNE-BIDS-Pipeline, covering the following steps:
1. Reads a single-subject raw recording (.edf or .fif file) and handles it like a BIDS structure (sub-01).
2. Applies filtering and resampling (band-pass/notch filters, optional resampling, etc.).
3. Detects breaks in continuous recordings (optional).
4. Extracts epochs around the events of interest.
5. Run spatial filters to detect and remove artifacts (eye-blinks and heartbeats).
   1. Run Signal Space Projections (SSP) for EOG/ECG artifacts.
   2. Run Independent Component Analysis (ICA) for EOG/ECG artifacts.
6. For MEG data only, optionally aplies Maxwell filtering (SSS/tSSS) including control over calibration/crosstalk files, movement compensation and empty-room-processing.
7. For EEG data only, allows setting the reference channels, bipolar channles and template montage.
8. Generates HTML reports summarizing each preprocessing step.

# Input

A single raw file (.edf for EEG or .fif for MEG) with a config.json generated by Brainlife that sets the pipeline parameters (filters cutoffs, ICA method, epoch time window, etc.).

# Output

- bids_input: the BIDS structure generated from the input file.
- out_dir: MNE-BIDS-Pipeline derivatives (cleaned epochs, SSP/ICA solutions and related output for the subject).
- html_report

# How to run this App on Brainlife 
The App runs inside a Singularity container built from ghcr.io/carlota-juarez/app-mne-bids:lite, based on python:3.11-slim with mne, mne-bids, and mne-bids-pipeline installed. These are the steps to run the App:
1. Register on Brainlife.io and create a project.
2. Upload a M/EEG single-subject recording (.edf or .fif)
3. Select this App from the App page and configure the provided parameters to suit the selected data.
4. Execute the App. Once completed, the cleaned/evoked data can be dwonloaded and passed to the Sensor Analysis App.
